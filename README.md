# Webpack Template with ESLint, Prettier, Husky & lint-staged

A modern frontend starter template configured with **Webpack 5**, **ESLint**, **Prettier**, **Husky**, and **lint-staged**.  
This setup ensures clean code formatting, automatic pre-commit checks, and a smooth development workflow.

## Features

- **Webpack 5** bundling (HTML, CSS, JS)
- **Development server** with live reloading
- **ESLint 9+** with the new `eslint.config.mjs` format
- **Prettier** for consistent code formatting
- **Husky** Git hooks for automated checks
- **lint-staged** to format only staged files
- **gh-pages** deployment support
- Modular project structure

## Tech Stack

- JavaScript (ES Modules)
- Webpack 5
- ESLint 9+
- Prettier 3
- Husky 9
- lint-staged 16

## Node.js Requirement

This project uses dependencies that require **Node.js 20 or higher**.

If you use Node 18, you may encounter errors like:

# SyntaxError: Invalid regular expression flags

To fix this, update Node.
